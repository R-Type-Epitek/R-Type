\doxysubsection{Detailed Description}
Manages a UDP server for handling network communication. 

The \doxylink{class_network_1_1_u_d_p_server}{UDPServer} class provides functionalities to create and manage a server that communicates using the UDP protocol. It handles client connections, data transmission, and room management for network clients.
