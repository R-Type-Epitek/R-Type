\doxysubsection{Detailed Description}
Manages a UDP server for handling network communication. 

The \mbox{\hyperlink{classNetwork_1_1UDPServer}{UDPServer}} class provides functionalities to create and manage a server that communicates using the UDP protocol. It handles client connections, data transmission, and room management for network clients.
