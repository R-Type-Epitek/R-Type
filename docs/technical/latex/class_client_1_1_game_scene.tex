\doxysubsection{Detailed Description}
Scene class specifically designed for game functionality in a client application. 

\doxylink{class_client_1_1_game_scene}{Game\+Scene} extends the IScene interface to provide functionalities specific to game scenes, including initialization of the ECS registry, entities, and custom systems. It also integrates with the network for game-\/related communications.
