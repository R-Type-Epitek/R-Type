\doxysubsection{Detailed Description}
Interface for handling network commands. 

\mbox{\hyperlink{classNetwork_1_1ICommandHandler}{ICommand\+Handler}} provides an interface for processing network commands received from clients. It includes methods for checking authorization and handling specific commands.
