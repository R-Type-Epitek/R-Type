\doxysubsection{Detailed Description}
A container class for managing scenes. 

The {\ttfamily \mbox{\hyperlink{classGameEngine_1_1Scene_1_1SceneContainer}{Scene\+Container}}} class allows adding, deleting, and retrieving scenes. It also provides iterators for iterating over the stored scenes.
